# WeatherApp

Приложение, показывающее прогноз погоды на день,3,неделю и выводящее погоду в данный момент.
Список поддерживаемых городов: Krasnodar, Murmansk, Madrid, NewYork, Moscow, Tanganrog, Vladivostok, Yekaterinburg
